# Ozymandias

This archive contains the original released source code for "Ozymandias" by Hugh Hancock / Strange Company. The original readme is preserved intact as `readme.txt`.

Praised by Roger Ebert, this adaption of Percy Shelleyâ€™s famous poem was one of the first completely game-independent pieces of Machinima. It was created using _Lithtech Film Producer_, one of the very first dedicated Machinima tools.
